# Security Policy

## Supported Versions

The following versions of `FaceRecog-Engine` are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

## Reporting a Vulnerability

We take the security of FaceRecog-Engine seriously. If you find a security vulnerability, please do not report it publicly (such as via a GitHub Issue). Instead, please report it via a private email or by creating a private vulnerability advisory on GitHub.

When reporting a vulnerability, please provide:
1. A descriptive title of the issue.
2. Steps to reproduce the issue or a proof-of-concept (PoC).
3. The potential impact of the vulnerability.

We will acknowledge your report within 48 hours and work with you to coordinate a security release.
